Developed a comprehensive iOS lost-and-found platform for university campuses using SwiftUI, featuring real-time item tracking with Google Maps integration, secure user authentication via Firebase, and a robust backend powered by Node.js. The application incorporates real-time chat functionality through WebSocket, cloud storage with AWS S3, and push notifications, enabling students to efficiently report, track, and recover lost items while maintaining data privacy and security.
